# multilingual

这个单词很明显是 multi + lingual，lingual 的意思是“related to speech or language (话语的；语言的)”，multilingual 就是“多语言的”，可以用来形容一个人会多门语言，或者一个环境是多语言的。

接下来几个单词都是老熟人了，只是简单过一过。

# lingustic

* [adjective] 语言学的

# lingustics

* [noun] 语言学

# linguist

* [noun] 语言学家；精通外国语言的人
